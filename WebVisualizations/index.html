Summary: Latitude vs. X

The purpose of this project was to analize how weather changes in relation to distance from the equator. To accomplish this analysis, I first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, I used Matplotlib to plot various aspects of the weather vs. latitude. Factors I looked at included: temperature, humidity, cloudiness, and wind speed. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
